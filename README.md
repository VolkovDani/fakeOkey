# Okey 101 - Multiplayer Browser Game

Браузерная мультиплеер-игра Okey 101 с WebSocket-соединением.

## Стек

- **Client**: React + TypeScript + Vite
- **Server**: Node.js + `ws` (WebSocket)
- **Shared**: общие типы и игровая логика

## Запуск

```bash
# Установить зависимости (из корня проекта)
npm install

# Запустить клиент и сервер одновременно
npm run dev

# Или раздельно:
npm run dev:client   # Vite dev server (http://localhost:5173)
npm run dev:server   # WebSocket server (ws://localhost:3001)
```

## Как играть

1. Открыть приложение в браузере (`http://localhost:5173`)
2. Ввести имя и создать комнату (выбрать 2/3/4 игрока) или ввести код существующей комнаты
3. Когда все игроки подключились, игра начинается автоматически

### Правила хода

1. **Взять тайл** -- из стопки (draw pile) или из сброса (discard pile)
2. **Выложить мелды** (необязательно) -- выбрать тайлы в руке и нажать "Meld"
3. **Сбросить тайл** -- выбрать один тайл и нажать "Discard"

### Типы мелдов

- **Set**: 3-4 тайла с одинаковым числом, разных цветов
- **Run**: 3+ последовательных тайлов одного цвета
- **Pair**: 2 одинаковых тайла (число + цвет)

### Opening (первый мелд)

- Для sets/runs: суммарная стоимость >= 101 очко
- Для pairs: минимум 5 пар

### Джокер

Индикаторный тайл определяет джокера: тот же цвет, число на 1 больше. Джокер может заменять любой тайл в set/run. Сброс джокера -- штраф 101 очко.

### Конец раунда

Раунд заканчивается, когда игрок сбросил последний тайл (рука пуста) или стопка закончилась. Проигравшие получают очки по стоимости оставшихся тайлов. Игра заканчивается, когда кто-то набирает 200+ очков.
